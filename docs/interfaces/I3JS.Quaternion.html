Interface Quaternion

Implementation of a quaternion. This is used for rotating things without incurring in the dreaded gimbal lock issue, amongst other advantages.

example

const quaternion = new THREE.Quaternion() : this; quaternion.setFromAxisAngle( new THREE.Vector3( 0, 1, 0 ), Math.PI / 2 ) : this; const vector = new THREE.Vector3( 1, 0, 0 ) : this; vector.applyQuaternion( quaternion ) : this;
