* A Track of quaternion keyframe values.

Constructor

[name]( [param:String name], [param:Array times], [param:Array values] )

* [name](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/String) identifier for the KeyframeTrack. * [times](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Array) array of keyframe times. * [values](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Array) values for the keyframes at the times specified, a flat array of quaternion components. * [interpolation](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Constant) the type of interpolation to use. See [Animation Constants](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Animation) for possible values. Default is [InterpolateLinear](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Animation).
