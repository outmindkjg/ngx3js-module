Texture Constants

Mapping Modes

* THREE.UVMapping THREE.CubeReflectionMapping THREE.CubeRefractionMapping THREE.EquirectangularReflectionMapping THREE.EquirectangularRefractionMapping THREE.CubeUVReflectionMapping THREE.CubeUVRefractionMapping * These define the texture's mapping mode. * [UVMapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Constant) is the default, and maps the texture using the mesh's UV coordinates. * The rest define environment mapping types. * [CubeReflectionMapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Constant) and [CubeRefractionMapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Constant) are for use with a [CubeTexture](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/CubeTexture), which is made up of six textures, one for each face of the cube. * [CubeReflectionMapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Constant) is the default for a [CubeTexture](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/CubeTexture). * [EquirectangularReflectionMapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Constant) and [EquirectangularRefractionMapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Constant) * are for use with an equirectangular environment map. Also called a lat-long map, an equirectangular texture represents a 360-degree view along the horizontal centerline, and a 180-degree view along the vertical axis, with the top and bottom edges of the image corresponding to the north and south poles of a mapped sphere. * See the [materials / envmaps](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_materials_envmaps) example.

Wrapping Modes

* THREE.RepeatWrapping THREE.ClampToEdgeWrapping THREE.MirroredRepeatWrapping * These define the texture's [wrapS](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Texture.wrapS) and [wrapT](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Texture.wrapT) properties, which define horizontal and vertical texture wrapping. * With [RepeatWrapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) the texture will simply repeat to infinity. * [ClampToEdgeWrapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) is the default. * The last pixel of the texture stretches to the edge of the mesh. * With [MirroredRepeatWrapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) the texture will repeats to infinity, mirroring on each repeat.

Magnification Filters

* THREE.NearestFilter THREE.LinearFilter * For use with a texture's [magFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Texture.magFilter) property, these define the texture magnification function to be used when the pixel being textured maps to an area less than or equal to one texture element (texel). * [NearestFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) returns the value of the texture element that is nearest (in Manhattan distance) to the specified texture coordinates. * [LinearFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) is the default and returns the weighted average of the four texture elements that are closest to the specified texture coordinates, and can include items wrapped or repeated from other parts of a texture, depending on the values of [wrapS](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Texture.wrapS) and [wrapT](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Texture.wrapT), and on the exact mapping.

Minification Filters

* THREE.NearestFilter THREE.NearestMipmapNearestFilter THREE.NearestMipmapLinearFilter THREE.LinearFilter THREE.LinearMipmapNearestFilter THREE.LinearMipmapLinearFilter * For use with a texture's [minFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Texture.minFilter) property, these define the texture minifying function that is used whenever the pixel being textured maps to an area greater than one texture element (texel). * In addition to [NearestFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) and [LinearFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant), the following four functions can be used for minification: * [NearestMipmapNearestFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) chooses the mipmap that most closely matches the size of the pixel being textured and uses the [NearestFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) criterion (the texel nearest to the center of the pixel) to produce a texture value. * [NearestMipmapLinearFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) chooses the two mipmaps that most closely match the size of the pixel being textured and uses the [NearestFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) criterion to produce a texture value from each mipmap. The final texture value is a weighted average of those two values. * [LinearMipmapNearestFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) chooses the mipmap that most closely matches the size of the pixel being textured and uses the [LinearFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) criterion (a weighted average of the four texels that are closest to the center of the pixel) * to produce a texture value. * [LinearMipmapLinearFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) is the default and chooses the two mipmaps that most closely match the size of the pixel being textured and uses the [LinearFilter](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) criterion to produce a texture value from each mipmap. The final texture value is a weighted average of those two values. * See the [materials / texture / filters](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_materials_texture_filters) example.

DDS / ST3C Compressed Texture Formats

* THREE.RGB_S3TC_DXT1_Format THREE.RGBA_S3TC_DXT1_Format THREE.RGBA_S3TC_DXT3_Format THREE.RGBA_S3TC_DXT5_Format * For use with a [CompressedTexture](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/CompressedTexture)'s [format](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Texture.format) property, these require support for the [WEBGL_compressed_texture_s3tc](https://www.khronos.org/registry/webgl/extensions/WEBGL_compressed_texture_s3tc/) * extension. * There are four [S3TC](https://en.wikipedia.org/wiki/S3_Texture_Compression) formats available via this extension.
