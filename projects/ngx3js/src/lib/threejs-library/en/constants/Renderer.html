WebGLRenderer Constants

Cull Face Modes

* THREE.CullFaceNone THREE.CullFaceBack THREE.CullFaceFront THREE.CullFaceFrontBack * [CullFaceNone](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) disables face culling. * [CullFaceBack](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) culls back faces (default). * [CullFaceFront](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) culls front faces. * [CullFaceFrontBack](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) culls both front and back faces.

Shadow Types

* THREE.BasicShadowMap THREE.PCFShadowMap THREE.PCFSoftShadowMap THREE.VSMShadowMap * These define the WebGLRenderer's [shadowMap.type](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/WebGLRenderer.shadowMap.type) property. * [BasicShadowMap](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) gives unfiltered shadow maps - fastest, but lowest quality. * [PCFShadowMap](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) filters shadow maps using the Percentage-Closer Filtering (PCF) algorithm (default). * [PCFSoftShadowMap](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) filters shadow maps using the Percentage-Closer Filtering (PCF) algorithm with better soft shadows especially when using low-resolution shadow maps. * [VSMShadowMap](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/constant) filters shadow maps using the Variance Shadow Map (VSM) algorithm. When using VSMShadowMap all shadow receivers will also cast shadows.

Tone Mapping

* THREE.NoToneMapping THREE.LinearToneMapping THREE.ReinhardToneMapping THREE.CineonToneMapping THREE.ACESFilmicToneMapping * These define the WebGLRenderer's [toneMapping](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/WebGLRenderer.toneMapping) property. * This is used to approximate the appearance of high dynamic range (HDR) on the low dynamic range medium of a standard computer monitor or mobile device's screen. * See the [WebGL / tonemapping](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_tonemapping) example.
