[name]

* A loader for loading a [Material](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Material) in JSON format. * This uses the [FileLoader](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/FileLoader) internally for loading files. * ### Code Example * ```javascript * // instantiate a loader * const loader = new THREE.MaterialLoader(); * // load a resource * loader.load( * // resource URL * 'path/to/material.json', * function ( material ) { // onLoad callback object.material = material; * }, * function ( xhr ) { // onProgress callback * console.log( (xhr.loaded / xhr.total * 100) + '% loaded' ); * }, * function ( err ) { // onError callback * console.log( 'An error happened' ); * } * );

Constructor

[name]( [param:LoadingManager manager] )

* @param manager - The [loadingManager](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/LoadingManager) for the loader to use. Default is [THREE.DefaultLoadingManager](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/LoadingManager). * Creates a new [name].

Properties

* See the base [Loader](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Loader) class for common properties.

* Object holding any textures used by the material. See *.setTextures*.

Methods

* See the base [Loader](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Loader) class for common methods.

[method:undefined load]( [param:String url], [param:Function onLoad], [param:Function onProgress], [param:Function onError] )

* @param url - the path or URL to the file. This can also be a [Data URI](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs). * @param onLoad - Will be called when load completes. The argument will be the loaded [Material](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Material). * @param onProgress - Will be called while load progresses. The argument will be the XMLHttpRequest instance, which contains .[total](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Integer) and .[loaded](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Integer) bytes. If the server does not set the Content-Length header; .[total](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Integer) will be 0. * @param onError - Will be called when load errors. * Begin loading from url.

[method:Material parse]( [param:Object json] )

* @param json - The json object containing the parameters of the Material. * Parse a JSON structure and create a new [Material](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Material) of the type [json.type](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/String) with parameters defined in the json object.

[method:this setTextures]( [param:Object textures] )

* @param textures - object containing any textures used by the material.
