[name]

* A simple caching system, used internally by [FileLoader](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/FileLoader).

Code Example

* To enable caching across all loaders that use [FileLoader](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/FileLoader), set THREE.Cache.enabled = true.

Examples

* [WebGL / geometry / text ](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_geometry_text) * [WebGL / interactive / instances / gpu](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_interactive_instances_gpu) * [WebGL / loader / ttf](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_loader_ttf)

Properties

[property:Boolean enabled]

* Whether caching is enabled. Default is *false*.

[property:Object files]

* An [object](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object) that holds cached files.

Methods

[method:undefined add]( [param:String key], [param:Object file] )

* @param key - the [key](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/String) to reference the cached file by. * @param file - The file to be cached. * Adds a cache entry with a key to reference the file. If this key already holds a file, it is overwritten.

[method:Any get]( [param:String key] )

* @param key - A string key * Get the value of [key](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/String). If the key does not exist *undefined* is returned.

[method:undefined remove]( [param:String key] )

* @param key - A string key that references a cached file. * Remove the cached file associated with the key.

[method:undefined clear]()

* Remove all values from the cache.
