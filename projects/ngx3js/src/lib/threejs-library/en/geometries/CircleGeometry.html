Constructor

[name]([param:Float radius], [param:Integer segments], [param:Float thetaStart], [param:Float thetaLength])

* @param radius - Radius of the circle, default = 1. * @param segments - Number of segments (triangles), minimum = 3, default = 8. * @param thetaStart - Start angle for first segment, default = 0 (three o'clock position). * @param thetaLength - The central angle, often called theta, of the circular sector. The default is 2*Pi, which makes for a complete circle.
