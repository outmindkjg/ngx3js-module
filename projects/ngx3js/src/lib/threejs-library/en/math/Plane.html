[name]

* A two dimensional surface that extends infinitely in 3d space, represented in [Hessian normal form](http://mathworld.wolfram.com/HessianNormalForm.html) * by a unit length normal vector and a constant.

Constructor

[name]( [param:Vector3 normal], [param:Float constant] )

* @param normal - a unit length [Vector3](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Vector3) defining the normal of the plane. Default is *(1, 0, 0)*. * @param constant - the signed distance from the origin to the plane. Default is *0*.

* @param matrix - the [Page:Matrix4] to apply. * @param optionalNormalMatrix - pre-computed normal [Page:Matrix3] of the Matrix4 being applied. * Apply a Matrix4 to the plane. The matrix must be an affine, homogeneous transform. * If supplying an [optionalNormalMatrix](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Matrix3), it can be created like so: * const optionalNormalMatrix = new THREE.Matrix3().getNormalMatrix( matrix );

[method:this set]( [param:Vector3 normal], [param:Float constant] )

Default is *0*. * Sets this plane's *.normal* and *.constant* properties by copying the values from the given normal.

[method:this setComponents]( [param:Float x], [param:Float y], [param:Float z], [param:Float w] )

* @param x - x value of the unit length normal vector. * @param y - y value of the unit length normal vector. * @param z - z value of the unit length normal vector. * @param w - the value of the plane's *.constant* property. * Set the individual components that define the plane.

[method:this setFromCoplanarPoints]( [param:Vector3 a], [param:Vector3 b], [param:Vector3 c] )

[a](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Vector3) - first point on the plane. [b](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Vector3) - second point on the plane. [c](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Vector3) - third point on the plane. * Defines the plane based on the 3 provided points. The winding order is assumed to be counter-clockwise, and determines the direction of the *.normal*.

[method:this setFromNormalAndCoplanarPoint]( [param:Vector3 normal], [param:Vector3 point] )

[method:this translate]( [param:Vector3 offset] )

* @param offset - the amount to move the plane by. * Translates the plane by the distance defined by the [offset](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Vector3) vector. * Note that this only affects the plane constant and will not affect the normal vector.
