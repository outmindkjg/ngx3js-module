[name]

* This class is used to convert a series of shapes to an array of [Path](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Path)s, for example an SVG shape to a path (see the example below). * ### Examples * [geometry / extrude / shapes2](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_geometry_extrude_shapes2)

Constructor

[name]( )

* Creates a new ShapePath. Unlike a [Path](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Path), no points are passed in as the ShapePath is designed to be generated after creation.

Properties

[property:Array subPaths]

* Array of [Path](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Path)s.

[property:Array currentPath]

* The current [Path](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Path) that is being generated.

[property:Color color]

* [Color](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Color) of the shape, by default set to white (0xffffff).

Methods

[method:this moveTo]( [param:Float x], [param:Float y] )

* Starts a new [Path](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Path) and calls [Path.moveTo](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Path.moveTo)( x, y ) on that [Path](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Path). * Also points [currentPath](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/ShapePath.currentPath) to that [Path](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Path).

[method:this lineTo]( [param:Float x], [param:Float y] )

* This creates a line from the [currentPath](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/ShapePath.currentPath)'s offset to X and Y and updates the offset to X and Y.

[method:this quadraticCurveTo]( [param:Float cpX], [param:Float cpY], [param:Float x], [param:Float y] )

* This creates a quadratic curve from the [currentPath](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/ShapePath.currentPath)'s offset to x and y with cpX and cpY as control point and updates the [currentPath](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/ShapePath.currentPath)'s offset to x and y.

[method:this bezierCurveTo]( [param:Float cp1X], [param:Float cp1Y], [param:Float cp2X], [param:Float cp2Y], [param:Float x], [param:Float y] )

* This creates a bezier curve from the [currentPath](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/ShapePath.currentPath)'s offset to x and y with cp1X, cp1Y and cp2X, cp2Y as control points and updates the [currentPath](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/ShapePath.currentPath)'s offset to x and y.

[method:this splineThru] ( [param:Array points] )

* @param points - An array of [Vector2](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Vector2)s Connects a new [SplineCurve](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/SplineCurve) onto the [currentPath](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/ShapePath.currentPath).

[method:Array toShapes]( [param:Boolean isCCW], [param:Boolean noHoles] )

* @param isCC - Changes how solids and holes are generated noHoles -- Whether or not to generate holes Converts the [subPaths](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/ShapePath.subPaths) array into an array of Shapes. By default solid shapes are defined clockwise (CW) and holes are defined counterclockwise (CCW). If isCCW is set to true, then those are flipped. If the parameter noHoles is set to true then all paths are set as solid shapes and isCCW is ignored.
