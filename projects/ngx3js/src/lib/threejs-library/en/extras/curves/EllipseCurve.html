[name]

* Creates a 2d curve in the shape of an ellipse. Setting the [xRadius](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Number) equal to the [yRadius](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Number) will result in a circle. * ### Code Example * ```javascript * const curve = new THREE.EllipseCurve( * 0, 0, // ax, aY * 10, 10, // xRadius, yRadius 0, 2 * Math.PI, // aStartAngle, aEndAngle false, // aClockwise 0 // aRotation ); * const points = curve.getPoints( 50 ); * const geometry = new THREE.BufferGeometry().setFromPoints( points ); * const material = new THREE.LineBasicMaterial( { color : 0xff0000 } ); * // Create the final object to add to the scene * const ellipse = new THREE.Line( geometry, material );

Constructor

[name]( [param:Float aX], [param:Float aY], [param:Float xRadius], [param:Float yRadius], [param:Radians aStartAngle], [param:Radians aEndAngle], [param:Boolean aClockwise], [param:Radians aRotation] )

* [aX](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Float) – The X center of the ellipse. Default is *0*. * [aY](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Float) – The Y center of the ellipse. Default is *0*. * [xRadius](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Float) – The radius of the ellipse in the x direction. Default is *1*. * [yRadius](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Float) – The radius of the ellipse in the y direction. Default is *1*. * [aStartAngle](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Radians) – The start angle of the curve in radians starting from the positive X axis. Default is *0*. * [aEndAngle](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Radians) – The end angle of the curve in radians starting from the positive X axis. Default is *2 x Math.PI*. * [aClockwise](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Boolean) – Whether the ellipse is drawn clockwise. Default is *false*. * [aRotation](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Radians) – The rotation angle of the ellipse in radians, counterclockwise from the positive X axis Default is *0*.
