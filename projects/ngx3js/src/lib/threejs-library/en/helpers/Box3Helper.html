[name]

* Helper object to visualize a [Box3](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Box3).

Code Example

* const box = new THREE.Box3(); * box.setFromCenterAndSize( new THREE.Vector3( 1, 1, 1 ), new THREE.Vector3( 2, 1, 3 ) ); * const helper = new THREE.Box3Helper( box, 0xffff00 ); * scene.add( helper );

Constructor

[name]( [param:Box3 box], [param:Color color] )

* [box](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Box3) -- the Box3 to show. * [color](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Color) -- the box's color. Default is 0xffff00. * Creates a new wireframe box that represents the passed Box3.

Properties

* See the base [LineSegments](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/LineSegments) class for common properties.

[property:Box3 box]

* The Box3 being visualized.

[method:undefined updateMatrixWorld]( [param:Boolean force] )

* This overrides the method in the base [Object3D](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object3D) class so that it also updates the wireframe box to the extent of the [.box](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Box3Helper.box) * property.
