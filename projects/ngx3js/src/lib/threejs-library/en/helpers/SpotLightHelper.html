[name]

* This displays a cone shaped helper object for a [SpotLight](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/SpotLight). * ### Code Example * ```javascript * const spotLight = new THREE.SpotLight( 0xffffff ); * spotLight.position.set( 10, 10, 10 ); * scene.add( spotLight ); * const spotLightHelper = new THREE.SpotLightHelper( spotLight ); * scene.add( spotLightHelper ); * ### Examples * [WebGL/ lights / spotlights ](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_lights_spotlights)

Constructor

[name]( [param:SpotLight light], [param:Hex color] )

* @param light - The [SpotLight](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/SpotLight) to be visualized. * @param color - if this is not the set the helper will take the color of the light.

Properties

* See the base [Object3D](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object3D) class for common properties.

[property:LineSegments cone]

* [LineSegments](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/LineSegments) used to visualize the light.

[property:SpotLight light]

* Reference to the [SpotLight](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/SpotLight) being visualized.

[property:Object matrix]

* Reference to the spotLight's [matrixWorld](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object3D.matrixWorld).

[property:Object matrixAutoUpdate]

* See [Object3D.matrixAutoUpdate](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object3D.matrixAutoUpdate). Set to *false* here as the helper is using the spotLight's [matrixWorld](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object3D.matrixWorld).

[property:hex color]

The color parameter passed in the constructor. Default is *undefined*. If this is changed, the helper's color will update the next time *.update* is called.

Methods

* See the base [Object3D](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object3D) class for common methods.

* Disposes of the light helper.

[method:undefined update]()

* Updates the light helper.
