[name]

* This displays a helper object consisting of a spherical [Mesh](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Mesh) for visualizing a [PointLight](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/PointLight). * ### Code Example * ```javascript * const pointLight = new THREE.PointLight( 0xff0000, 1, 100 ); * pointLight.position.set( 10, 10, 10 ); * scene.add( pointLight ); * const sphereSize = 1; * const pointLightHelper = new THREE.PointLightHelper( pointLight, sphereSize ); * scene.add( pointLightHelper ); * ### Examples * [WebGL / helpers](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_helpers)

Constructor

[name]( [param:PointLight light], [param:Float sphereSize], [param:Hex color] )

* @param light - The light to be visualized. * @param sphereSize - The size of the sphere helper. Default is *1*. * @param color - if this is not the set the helper will take the color of the light.

Properties

* See the base [Mesh](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Mesh) class for common properties.

[property:PointLight light]

* The [PointLight](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/PointLight) that is being visualized.

[property:Object matrix]

* Reference to the pointLight's [matrixWorld](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object3D.matrixWorld).

[property:Object matrixAutoUpdate]

* See [Object3D.matrixAutoUpdate](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object3D.matrixAutoUpdate). Set to *false* here as the helper is using the pointLight's [matrixWorld](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Object3D.matrixWorld).

[property:hex color]

The color parameter passed in the constructor. Default is *undefined*. If this is changed, the helper's color will update the next time *.update* is called.

Methods

* See the base [Mesh](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Mesh) class for common methods.

[method:undefined dispose]()

* Dispose of the pointLightHelper.

[method:undefined update]()

* Updates the helper to match the position of the *.light*.
