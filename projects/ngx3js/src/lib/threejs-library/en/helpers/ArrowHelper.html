[name]

* An 3D arrow object for visualizing directions.

Code Example

* const dir = new THREE.Vector3( 1, 2, 0 ); * // normalize the direction vector (convert to vector of length 1) * dir.normalize(); * const origin = new THREE.Vector3( 0, 0, 0 ); * const length = 1; * const hex = 0xffff00; * const arrowHelper = new THREE.ArrowHelper( dir, origin, length, hex ); * scene.add( arrowHelper );

Examples

* [WebGL / shadowmesh](https://outmindkjg.github.io/ngx3js-doc/#/examples/webgl_shadowmesh)

Constructor

[name]([param:Vector3 dir], [param:Vector3 origin], [param:Number length], [param:Number hex], [param:Number headLength], [param:Number headWidth] )

* [dir](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Vector3) -- direction from origin. Must be a unit vector. * [origin](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Vector3) -- Point at which the arrow starts. * [length](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Number) -- length of the arrow. Default is *1*. * [hex](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Number) -- hexadecimal value to define color. Default is 0xffff00. * [headLength](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Number) -- The length of the head of the arrow. Default is 0.2 * length. * [headWidth](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Number) -- The width of the head of the arrow. Default is 0.2 * headLength.
