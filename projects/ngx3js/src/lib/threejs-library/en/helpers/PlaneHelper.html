[name]

* Helper object to visualize a [Plane](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Plane). * ### Code Example * ```javascript * const plane = new THREE.Plane( new THREE.Vector3( 1, 1, 0.2 ), 3 ); * const helper = new THREE.PlaneHelper( plane, 1, 0xffff00 ); * scene.add( helper );

Constructor

[name]( [param:Plane plane], [param:Float size], [param:Color hex] )

* @param plane - the plane to visualize. * @param size - side length of plane helper. Default is 1. * @param color - the color of the helper. Default is 0xffff00. * Creates a new wireframe representation of the passed plane.

Properties

* See the base [LineSegments](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/LineSegments) class for common properties.

[property:Plane plane]

* The [plane](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Plane) being visualized.

[property:Float size]

* The side lengths of plane helper.
